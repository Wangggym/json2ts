# Preparation

- node16+

# Install and use

### Get the project code

```
git clone https://github.com/brain/micro-apps
git checkout develop
cd clip
```

### Installation dependencies

```
yarn
```

### Run

```
yarn start
```

Open [http://localhost:12640](http://localhost:3000) to view it in the browser.

### Build

```
yarn build
```

### ESLint

```
yarn fix
```
